# Reading the Grid

A convolutional neural network (CNN) that detects faults in solar-panel infrastructure
from **electroluminescence (EL) images**, paired with a **Grad-CAM** heatmap that shows
*where* the model is looking when it makes a prediction.

> ⚡ The goal is not just a fault/no-fault label, but an interpretable one — so a human
> inspector can see the region of the panel that drove the decision.

## Setup

Built and tested on a **MacBook Air (Apple Silicon)**. PyTorch uses the **MPS** (Metal)
backend for GPU acceleration on Apple Silicon.

```bash
# 1. Clone the repo
git clone <your-repo-url>
cd "Reading the Grid"

# 2. Create and activate a virtual environment
python3 -m venv venv
source venv/bin/activate          # macOS / Linux

# 3. Install dependencies
pip install --upgrade pip
pip install -r requirements.txt
```
